Objetivo: Desenvolvimento de um sistema para automatizar o processo de atendimento aos pacientes.

Overview do fluxo do processo:
  - O processo inicia com a inclusão de um Paciente na agenda para receber a 1ª consulta espiritual.
  - Para a 1ª consulta, no momento do Check-in, a atendente irá cadastrar o paciente no sistema.
  - Para os pacientes já agendados, a atendente irá efetuar somente o Check-in.
  - A cada check-in realizado na agenda, o sistema irá atualizar automaticamente a lista de consulta/tratamento espiritual e a lista de banho de luz/ bastão. Para sequenciar as listas de atendimentos, o sistema irá considerar as recomendações registradas nos apontamentos de cada paciente, assim como irá aplicar as regras de prioridades e a ordem do check-in .
  - Após cada consulta espiritual, a atendente irá checar e se necessário irá atualizar o cadastro do paciente com a inclusão de novas recomendações.
  - A cada atualização no cadastro do paciente o sistema irá automaticamente as agendas.

Módulos do sistema:
  1. Pacientes
    - Manutenção de Cadastro
    - Histórico de atendimentos

  2. Agenda
    - Consulta espiritual
    - Banho de Luz / bastão

  3. Lista de atendimentos
    - Consulta espiritual
    - Banho de Luz / bastão

1. Modulo pacientes
  - Este módulo tem como objetivo registrar os dados do paciente e as recomendações.
  - Este módulo permite a inclusão de um novo paciente, consultar e alterar dados de cadastro dos pacientes.
  - A aba de inclusão irá disponibilizar na tela os campos para o preenchimento.
  - As abas de consulta / alteração irão disponibilizar a lista de pacientes cadastrados, permitindo a pesquisa pela primeira(s) letra(s) , nome , etc.
  - O sistema irá manter na manutenção de cadastro somente o último atendimento ou o paciente em atendimento.
  - Sempre que iniciar um novo tratamento, o sistema irá mover o tratamento anterior para o histórico, disponibilizando a consulta quando necessário.

  O módulo Pacientes será composto por dois submódulos com as opções abaixo:
    - Manutenção de Cadastro
    - Aba de Inclusão
    - Aba de Alteração
    - Aba de Consulta
    - Histórico de atendimentos
    - Consulta

2. Modulo agendamento
  Com exceção do primeiro agendamento, o preenchimento da agenda será gerado pelo sistema com base nas informações parametrizadas no cadastro do paciente.
  Para o primeiro agendamento, o sistema irá disponibilizar uma opção na aba Consulta espiritual para incluir manualmente o nome do novo paciente.
  Para verificar as agendas de “Consulta espiritual” e “Banho de luz”/ “bastão”, o sistema irá disponibilizar um calendário no PopUp de cada agenda para selecionar a data da agenda e realizar a consulta.

  Agenda
    - Consulta espiritual
      - 1º agendamento
    - Banho de Luz / bastão

  O sistema deverá gerar a agenda com as seguintes regras:
    - Agendamento somente às terças-feiras, exceto nos feriados e na última quinzena de dezembro.
    - Deverá ser aberto duas agendas, uma para consulta/tratamento espiritual e outra para o banho de luz / bastão
    - Selecionar os pacientes em atendimento (status=T) com os campos Tratamento espiritual = “S” e/ou Banho de luz/bastão = “S”
    - Se o tratamento espiritual for igual a “S” preencher somente a data da próxima agenda
    - Se o banho de luz e/ou bastão igual a “S”, calcular conforme segue:
      - Se o paciente tiver somente um local, agendar o paciente até a data de término
      - Se o paciente tiver mais que um local agendar o paciente com a maior data de término.
    - Opções no agendamento:
      - Permitir check-in do paciente com data menor ou igual a data corrente
        - Atualiza a lista de atendimento das consultas/tratamento espiritual e banho de luz/ bastão
      - Encerramento da agenda
        - Atualiza o status do paciente com OK ou Faltou
        - O sistema deverá reagendar o banho de luz na ausência do paciente após a última data de banho pré-agendada.
        - Duas faltas consecutivas do paciente, limpar as agendas do banho de luz/bastão, mover a última data de banho de luz / bastão na data de término e cancelar o agendamento do atendimento espiritual.

  Observações:
    - Quando a ausência é comunicada antecipadamente, após duas faltas consecutivas o paciente deverá solicitar um agendamento para uma nova consulta espiritual.
    - Quando as ausências não são comunicadas, após a segunda falta, o paciente deverá retornar para o atendimento de quinta-feira para receber um encaminhamento para os tratamentos de terça-feira.

3. Modulo lista de atendimentos na terça-feira
  Este módulo tem como objetivo gerar uma lista para o sequenciamento do banho de luz / bastão e uma lista para consulta/tratamento espiritual.

  A lista de atendimento será gerada automaticamente pelo sistema considerando a relação de pacientes da agenda, as prioridades e a sequência de chegada registrada no check-in de cada paciente.

    - Atendimento
      - Consulta espiritual
      - Banho de Luz/ bastão

    As regras de cada módulo serão especificadas a seguir:

      - As listas são atualizadas quando é efetuado o check-in do paciente.
      - Os pacientes com prioridade “E” (Exceção) tem prioridade em relação a “I” (Idoso) e os demais
      - O paciente que recebe banho de luz e tratamento espiritual tem prioridade na lista do banho de luz/bastão.
